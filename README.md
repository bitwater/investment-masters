# Investment Masters —— AI 投资大师委员会

一个 Skill，召唤六位顶级投资大师组成虚拟委员会，对任意标的进行独立分析和平权投票，输出可执行的投资裁决报告。

## 为什么做这个？

投资决策最怕单一视角盲区。这个系统把六种截然不同的投资哲学——价值投资、周期研判、本分哲学、宏观交易、全天候配置、反脆弱思维——强制并行运行，互相看不到对方的结论，最后用规则整合成一份裁决。

本质上是用 AI 模拟一个高质量的投资委员会流程。

## 六位投资大师

| 委员 | 核心视角 | 擅长捕捉 |
|------|---------|---------|
| **查理·芒格** | 护城河 / 生意本质 / 能力圈 | 长期竞争优势、管理层质量、估值安全边际 |
| **霍华德·马克斯** | 周期位置 / 情绪温度 / 第二层思维 | 市场过热/过冷信号、风险不对称性 |
| **段永平** | 本分 / 动机审查 / 十年持有 | 生意是否真正理解、买入动机是否纯粹 |
| **德鲁肯米勒** | 宏观流动性 / 催化剂 / 止损纪律 | 宏观拐点、流动性变化、交易时机 |
| **雷·达利欧** | 全天候 / 债务周期 / 原则决策 | 宏观政体判断、压力测试、组合韧性 |
| **纳西姆·塔勒布** | 反脆弱 / 杠铃策略 / 尾部风险 | 黑天鹅暴露、收益凸性、隐藏脆弱性 |

六位大师平权（各 ~16.7%），独立分析，互不可见。

## 三种分析模式

**买入判断** —— "分析腾讯"、"要不要买英伟达"、"帮我看看 BTC"

**仓位管理** —— "我持有半仓 BTC，怎么操作"、"已持有 NVDA，加仓还是减仓"

**比较分析** —— "腾讯和谷歌选哪个"

## 使用方法

### 触发

在对话中直接说：

```
投资大师怎么看英伟达
委员会分析腾讯
投资大师腾讯和谷歌选哪个
```

Skill 会自动匹配并启动分析流程。

### 支持的资产类型

- **股票**：A 股、港股、美股
- **ETF**
- **加密资产**：BTC 等
- **大宗商品**：黄金等

每位大师用自己的通用分析框架中性评估任何资产类型，不预设立场。

## 执行流程

```
数据准备 → 识别模式 → 并行召唤六位委员 → 整合裁决 → 输出报告
```

1. **数据准备**：通过 `scripts/fetch_price.py` 抓取实时行情（主数据源：腾讯 qt.gtimg.cn，备用：stooq.com），再用搜索补充基本面数据
2. **识别模式**：根据用户请求自动判断是买入判断、仓位管理还是比较分析
3. **并行分析**：同时启动 6 个独立 sub-agent，各自按对应大师的框架输出评分和判断
4. **整合裁决**：汇总六份独立意见，计算加权综合分，检查否决规则，输出最终建议

## 评级体系

| 综合得分 | 评级 | 建议仓位 |
|---------|------|---------|
| 8.0 – 10.0 | ⭐⭐⭐⭐⭐ 强烈买入 | 15–25% |
| 6.5 – 7.9 | ⭐⭐⭐⭐ 买入（保守） | 8–15% |
| 5.0 – 6.4 | ⭐⭐⭐ 等待/观望 | 0–5% |
| 3.5 – 4.9 | ⭐⭐ 谨慎 | 0% |
| < 3.5 | ⭐ 回避 | 0% |

### 否决规则

- **硬否决**：≥ 3 位委员评分 ≤ 4 → 直接回避，0% 仓位
- **软否决**（可叠加）：
  - 马克斯评分 ≤ 5 且市场情绪 = 贪婪 → 建议仓位减半
  - 段永平动机审查不通过 → 仓位上限 30%
  - 达利欧评分 ≤ 4 且处于债务周期危险阶段 → 仓位减半
  - 塔勒布判断存在致命尾部风险 → 仓位上限 30%

## 设计理念

- **独立性**：六个 sub-agent 并行运行，互相不可见结论，避免群体思维
- **多样性**：价值派、周期派、本分派、宏观派、全天候派、反脆弱派——六种正交视角
- **资产中性**：每位大师用自己的通用框架分析，不对任何资产类型预设立场
- **可执行**：输出不是"建议关注"这种废话，而是具体的仓位比例、止损价、加仓条件
- **有纪律**：否决规则是硬编码的，不会因为情绪被覆盖
