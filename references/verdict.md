# 委员会裁决整合规则 v3

## 加权评分（各 ~16.7%）

综合得分 = (芒格 + 马克斯 + 段永平 + 德鲁肯米勒 + 达利欧 + 塔勒布) / 6

## 评级体系

| 综合得分 | 评级 | 建议仓位 |
|---|---|---|
| 8.0–10.0 | ⭐⭐⭐⭐⭐ 强烈买入 | 15–25% |
| 6.5–7.9 | ⭐⭐⭐⭐ 买入（保守） | 8–15% |
| 5.0–6.4 | ⭐⭐⭐ 等待/观望 | 0–5%（等回调） |
| 3.5–4.9 | ⭐⭐ 谨慎 | 0%（观察名单） |
| <3.5 | ⭐ 回避 | 0% |

> 仓位为相对仓位（占该资产类别配置比例），非绝对金额。

## 否决规则

**硬否决**：≥3 位委员评分 ≤ 4/10 → 评级「回避」，仓位 0%。

**软否决（各自独立触发，叠加生效）**：
- 马克斯评分 ≤ 5 且情绪温度=「贪婪/极度贪婪」→ 仓位减半
- 段永平动机审查不通过 → 仓位上限 10%
- 达利欧评分 ≤ 4 且判断处于债务周期危险阶段 → 仓位减半
- 塔勒布判断存在致命尾部风险 → 仓位上限 10%，或要求杠铃对冲

## 满仓条件

五位或以上委员评分 ≥ 7 → 可取该评级建议仓位区间的上限值。

## 冲突处理

| 冲突场景 | 处理方式 |
|---|---|
| 芒格看好 + 马克斯说贵 | 买入但减半，等回调 |
| 段永平动机不通过 + 其他人看好 | 动机优先，放弃或极小仓位 |
| 德鲁肯米勒说时机不对 | 列入观察，等催化剂明确 |
| 达利欧判断宏观环境不匹配 | 减仓，对冲宏观敞口 |
| 塔勒布判断有尾部风险 + 其他人看好 | 控制仓位上限，定义最大亏损 |
| 价值派（芒格+段永平）vs 宏观派（达利欧+德鲁肯米勒）| 小仓位，设明确加仓触发条件 |
| 3:3 完全分裂 | 默认保守：观望或极小仓位（≤3%），等分歧收敛 |

## 报告必含要素

1. 六委员各自评分 + 核心判断（1-2 句）
2. 加权综合得分（保留 1 位小数）
3. 最终评级
4. 建议仓位 %
5. 核心逻辑（1-2 句共识）
6. 主要风险（1 句，最悲观委员的核心担忧）
7. 止损条件（1 句，来自德鲁肯米勒；若塔勒布更保守，取更保守的）
8. 尾部风险评估（1 句，来自塔勒布）
9. 最大分歧：[大师A] X分 vs [大师B] Y分，分歧焦点（1 句）
10. 若触发硬否决 / 软否决，明确标注
